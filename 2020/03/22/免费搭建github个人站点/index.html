    感谢伟大的互联网，让我有了免费建个人网站的机会

    这里记录了想建站的缘由，建站过程参考网上各路大神的分享

    概述

    GitHub Pages is designed to host your personal, organization, or project pages from a GitHub repository.

    GitHub Pages可以免费把项目中的静态文件托管起来，一句话概括用它搭建个人站点过程，把静态网页生成好，上传到这个项目里，剩下的交给GitHub Pages。

    本文涉及的两个关键步骤：

    1. github配置
    2. 静态网页生成工具Hexo的使用

      github配置

    更进一步

    主题

    我们使用的是Hexo默认的主题，事实上Hexo支持很多种主题供我们进行个性化的设置，在各主题的Github页面中都会有详细的安装方法，有个壁坑技巧需要分享：有些主题是有bug的，在生成静态页面的时候会报错。而配置主题是比较耗时间的，所以建议在把主题下载下来之后，先执行hexo g && hexo s生成页面测试一下没问题后再做主题的其他详细配置

    配图

    Github存储空间有限，国内访问慢。所以图片存储建议选用图床，我现在使用的图床服务是路过，简单好用，空间大，速度还很快，免费，推荐。
